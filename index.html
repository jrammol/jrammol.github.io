Inicio

Este manual está diseñado para guiarte a través de todas las funcionalidades de iPortfolio, una aplicación móvil desarrollada para ayudarte a gestionar tus carteras de valores de manera eficiente y efectiva. En este documento encontrarás instrucciones detalladas sobre cómo registrarte, iniciar sesión, gestionar tus inversiones, y aprovechar al máximo todas las herramientas y funcionalidades que ofrece la aplicación.

Objetivo del Manual

El objetivo de este manual es proporcionar una guía completa y fácil de seguir para que puedas utilizar iPortfolio con confianza. Ya seas un inversor principiante o experimentado, este manual te ayudará a familiarizarte con todas las características de la aplicación, desde la creación de tu cuenta hasta la gestión avanzada de tus carteras de valores.

Registro en la aplicación

El registro de usuario permite a un nuevo usuario crear una cuenta en la aplicación iPortfolio proporcionando sus datos personales y de contacto.

Paso a Paso para Registrar un Nuevo Usuario

1. Abrir la Aplicación

Inicia la aplicación iPortfolio desde tu dispositivo Android.

2. Navegar a la Pantalla de Registro

En la pantalla de inicio, selecciona la opción "Registro" para acceder al formulario de registro.

3. Completar el Formulario de Registro

Introduce tus datos en los campos correspondientes:

  • Nombre: Introduce tu nombre completo.
  • Apellido: Introduce tu apellido.
  • Teléfono: Introduce tu número de teléfono (debe tener 9 dígitos).
  • Correo Electrónico: Introduce una dirección de correo electrónico válida.
  • Nombre de Usuario: Introduce un nombre de usuario.
  • Contraseña: Introduce una contraseña segura.
  • Confirmar Contraseña: Introduce nuevamente la contraseña para confirmarla.

4. Validar los Datos Ingresados

Asegúrate de que todos los campos estén completos y correctamente llenados:

  • Verifica que el correo electrónico tenga un formato válido.
  • Asegúrate de que el número de teléfono tenga exactamente 9 dígitos.
  • Asegúrate de que las contraseñas coincidan.

5. Enviar el Formulario

Pulsa el botón "Registrar" para enviar tus datos y crear tu cuenta. Si algún campo no es válido, la aplicación mostrará un mensaje de error indicando el problema específico.

6. Confirmación de Registro

Si los datos son válidos y el registro es exitoso, recibirás una notificación en pantalla confirmando que la cuenta ha sido creada correctamente. Serás redirigido automáticamente a la pantalla de inicio de sesión para acceder a la aplicación con tu nueva cuenta.

Solución de Problemas Comunes durante el Registro

Correo electrónico inválido

Asegúrate de introducir una dirección de correo electrónico válida. La aplicación verifica que el formato del correo electrónico sea correcto.

Número de teléfono inválido

Asegúrate de que el número de teléfono tenga 9 dígitos.

Contraseñas no coinciden

Asegúrate de que la contraseña y la confirmación de la contraseña sean idénticas.

Campos vacíos

Todos los campos del formulario deben ser completados. Si algún campo está vacío, la aplicación mostrará un mensaje de error solicitando que se complete.
